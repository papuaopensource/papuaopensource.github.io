Prinsip Kami

Tiga prinsip utama yang menjadi landasan komunitas Papua Open Source

Share

Mendistribusikan kode sumber ke publik, membuka akses pengetahuan untuk semua.

Improve

Modifikasi atau perbaikan kode untuk kualitas lebih baik, terus mengembangkan solusi.

Repeat

Teruskan siklus berbagi dan perbaikan untuk perkembangan berkelanjutan.

Cara Berkontribusi

Ada banyak cara untuk berkontribusi pada proyek-proyek Papua Open Source

Kontribusi Kode
Bantu pengembangan proyek dengan kontribusi kode

Berikut adalah langkah-langkah untuk berkontribusi kode pada proyek kami:

  1. Fork repositori proyek dari GitHub kami
  2. Clone repositori ke komputer lokal Anda
  3. Buat branch baru untuk fitur atau perbaikan yang ingin Anda kerjakan
  4. Kerjakan perubahan dan commit dengan pesan yang jelas
  5. Push perubahan ke repositori fork Anda
  6. Buat Pull Request ke repositori utama
Lihat Repositori GitHub

Mulai Berkontribusi

Tidak perlu menjadi seorang ahli untuk mulai berkontribusi. Kami menyambut kontributor dari semua tingkat keahlian, dari pemula hingga profesional.

Kami telah menandai beberapa issue di GitHub dengan label "good first issue" yang cocok untuk kontributor pertama kali. Anda juga dapat bergabung dengan diskusi di forum kami untuk mendapatkan bantuan dan bimbingan.
