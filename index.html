Komunitas Open Source

Kolaborasi Open Source untuk Tanah Papua

Papua Open Source memfasilitasi kontribusi bersama memajukan perkembangan teknologi di Tanah Papua melalui kolaborasi berbasis Open Source.

Prinsip Kami

Tiga prinsip utama yang menjadi landasan komunitas Papua Open Source

Share

Mendistribusikan kode sumber ke publik, membuka akses pengetahuan untuk semua.

Improve

Modifikasi atau perbaikan kode untuk kualitas lebih baik, terus mengembangkan solusi.

Repeat

Teruskan siklus berbagi dan perbaikan untuk perkembangan berkelanjutan.
